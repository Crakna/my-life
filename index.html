html -> cuida de todo conteudo textual.
css -> cuida de todo conteudo visual.
java script -> cuida de todos os elementos interativos.
Sites |Front-end| são site que rodam direto no seu navegador, esse são sites leves.
Sites |Back-end| são sites que rodam direto do servidor, esse são sites pesados.
Sites |Estaticos| são sites que não precisam de um banco de dados para contruir o pagina, o HTML, css e o js,
são enviados para seu computador e assim a pagina é criada em seu navegador.

@Atributos@
[style=] Permite incluir elementos CSS (estilos) dentro da tag.
[lang=] Define o idioma principal do elemento;
[title=] Define o título do elemento.
[alt=] Define um texto alternativo e, por isso, é muito utilizado em imagens, auxilia nas práticas de SEO.
[hidden] Oculta o elemento.
[align=] Permite definir o padrão de alinhamento
desse elemento, como por exemplo: right, center,left e justify.
[width=] Define uma largura para o elemento.
[height=] Define uma altura para o elemento.
